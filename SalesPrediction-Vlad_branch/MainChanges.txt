Настройка CI/CD для микросервиса прогнозирования продаж

Проделанная работа:
1. Создан docker-compose.yml для автоматического деплоя микросервиса
2. Создан build_and_run.sh скрипт с поддержкой веток для CI/CD
3. Добавлена система маркировки контейнеров по веткам
4. Настроен автоматический деплой с очисткой старых контейнеров
5. Добавлена проверка здоровья сервиса (healthcheck)

Микросервис запускается на порту 8000 и содержит:
- microservice.py - основной код FastAPI сервиса
- gbr_pipeline.pkl - модель машинного обучения
- historical_data.csv - исторические данные
- sku_metrics.csv - метрики SKU

Деплой выполняется автоматически при push в GitHub с помощью GitHub Actions.
